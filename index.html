Fasterapple

The BIND database for a DNS service in China Mainland to accelerate the speed of Apple App Store. Not being updated frequently since Apple set up an App Store mirror in China Mainland.

Download this project as a .zip file Download this project as a tar.gz file

English version of README is below.

请将你的DNS服务器设置为42.121.236.241和74.117.62.158直接使用DNS服务

You can set your DNS setting to 42.121.236.241,74.117.62.158 to use the service directly

说明

db文件夹中的配置文件为BIND9所使用。里面包含了所有FasterApple项目解析所使用的配置文件。欢迎参考、借鉴、使用。 请点击上方的.zip或.tar.gz下载相应格式的配置文件压缩包。 可直接在 named.conf 中使用 include 将 db/zones.conf 包含进去即可。

All the BIND9 configuration files that the FasterApple project has used are in db directory. Please click .tar.gz or .zip above to download all the files in the corresponding package format. You can just simply use include to put db/zones.conf into your named.conf
